# 🌸 Pollinations Image Generator

A clean, browser-based UI for generating images via the [Pollinations.ai](https://pollinations.ai) API.

## Features

- 🔑 Secure API key storage (never transmitted in plaintext logs)
- 🖼️ Multiple AI models: Flux, Flux-Realism, Flux-3D, Flux-Anime, Flux-Pro, Turbo
- 📐 Aspect ratio presets + custom width/height
- 🎲 Seed control for reproducible results
- ✨ AI prompt enhancement toggle
- ⬇️ One-click image download

## Getting Started

1. Get your API key at [enter.pollinations.ai](https://enter.pollinations.ai)
2. Open `index.html` in your browser
3. Enter your API key on the setup screen — it's stored locally and never shared
4. Start generating images!

## Usage

1. Enter a descriptive prompt
2. Choose a model and dimensions
3. Click **Generate Image**
4. Download the result

## Tech Stack

Plain HTML · CSS · Vanilla JavaScript · Pollinations.ai REST API
